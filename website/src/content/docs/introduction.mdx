## What is OpenWarp?

<p data-lang="en">OpenWarp is a Rust-based terminal that reuses the upstream Warp UI — <code>Blocks</code>, <code>Workflows</code>, command suggestions, the keymap layer — and replaces the closed AI gateway with explicit, local, editable plumbing. It is <strong>not</strong> the official Warp product, and it is not affiliated with Warp Inc.</p>
<p data-lang="zh">OpenWarp 是基于 Rust 的终端,复用上游 Warp 的 UI ── <code>Blocks</code>、<code>Workflows</code>、命令建议、keymap 层 ── 把封闭的 AI gateway 替换成显式、本地、可编辑的管线。它<strong>不是</strong>官方 Warp,也与 Warp Inc. 无任何关联。</p>

## Why a fork?

<p data-lang="en">Upstream Warp routes AI traffic through its own gateway. That works for most people, but it doesn't work for users who want to bring their own keys, point at a self-hosted model, or audit the system prompt that's getting sent on their behalf. OpenWarp changes <em>five</em> things; everything else stays.</p>
<p data-lang="zh">上游 Warp 把 AI 流量经过自己的 gateway。对大部分人没问题,但对要自带 key、指向自托管模型、审计系统提示词的用户来说不够。OpenWarp 只改 <em>五件事</em>,其他全留。</p>

## Install

<p data-lang="en">There are no binary installers yet — OpenWarp is in early development. Build from source via either the bootstrap script or cargo directly.</p>
<p data-lang="zh">目前没有发布安装包 ── OpenWarp 处于早期开发阶段。请从源码构建,推荐使用 bootstrap 脚本,也可以直接用 cargo。</p>

## First run

<p data-lang="en">After launch, open <code>Settings → Agents → Providers</code> and add at least one provider. The minimal config looks like this:</p>
<p data-lang="zh">启动后,打开 <code>设置 → Agents → 提供商</code>,至少添加一个 provider。最小配置如下:</p>

<CodeCard
  file="~/.config/openwarp/settings.toml"
  lang="TOML"
  html={`<span class="com"># Pick any OpenAI-compatible endpoint</span>
<span class="key">[providers.deepseek]</span>
base_url = <span class="str">"https://api.deepseek.com/v1"</span>
protocol = <span class="str">"deepseek"</span>     <span class="com"># or: openai, anthropic, gemini, ollama</span>
api_key  = <span class="str">"sk-•••"</span>          <span class="com"># stored locally, never sent home</span>

<span class="key">[[providers.deepseek.models]]</span>
display = <span class="str">"DeepSeek V4 Pro"</span>
id      = <span class="str">"deepseek-v4-pro"</span>
context = <span class="num">1000000</span>
output  = <span class="num">384000</span>`}
/>

<p class="muted" data-lang="en">Hit <code>⌘P</code> to switch model, <code>/</code> to open the slash menu, <code>⌃⇧+</code> to enter code review.</p>
<p class="muted" data-lang="zh">按 <code>⌘P</code> 切换模型,<code>/</code> 打开 slash 菜单,<code>⌃⇧+</code> 进入代码评审。</p>

## SSH manager

<p data-lang="en">OpenWarp adds a first-class SSH manager — net-new on top of upstream. Hosts live in their own file, jump-host chains are supported, and the remote session renders as Blocks just like a local shell.</p>
<p data-lang="zh">OpenWarp 在上游基础上新增了一等公民的 SSH 管理器。主机定义放在独立文件,支持跳板机链路,远端会话同样以 Blocks 形式呈现,体验与本地 shell 一致。</p>

<CodeCard
  file="~/.config/openwarp/ssh.toml"
  lang="TOML"
  html={`<span class="key">[host.bastion]</span>
user = <span class="str">"ops"</span>
host = <span class="str">"bastion.example.com"</span>
port = <span class="num">22</span>

<span class="key">[host.prod-edge]</span>
user = <span class="str">"deploy"</span>
host = <span class="str">"edge-01.lan"</span>
jump = <span class="str">"bastion"</span>           <span class="com"># chain through bastion</span>
forward_agent = <span class="acc">true</span>`}
/>

<div class="callout warn">
  <svg class="ico" viewBox="0 0 16 16" fill="none" stroke="currentColor" stroke-width="1.8"><path d="M8 2l6 11H2z"/><path d="M8 6v3M8 11h.01"/></svg>
  <div>
    <span data-lang="en">SSH credentials are read from your existing <code>{'~/.ssh'}</code> — OpenWarp does not duplicate them or upload them anywhere. The <code>ssh.toml</code> file holds <em>connection metadata only</em>.</span>
    <span data-lang="zh">SSH 凭证沿用你现有的 <code>{'~/.ssh'}</code> ── OpenWarp 不复制、不上传。<code>ssh.toml</code> 只保存<em>连接元数据</em>。</span>

## Localization

<p data-lang="en">English and Simplified Chinese ship in-tree. The locale switcher is in <code>Settings → Appearance → Language</code>; restart OpenWarp for the change to take full effect.</p>
<p data-lang="zh">英文与简体中文随发行包一起提供。语言切换位于 <code>设置 → 外观 → 语言</code>;切换后需要重启 OpenWarp 完全生效。</p>
